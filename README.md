# NodeJS-TestApp
Aplicaciones de prueba en Node.js

## App simple Node.js
	cd ~
	git clone https://github.com/wnpower/NodeJS-TestApp.git
	cd NodeJS-TestApp
	npm install
	node test_app.js
## App Node.js con Websockets
	cd ~
	git clone https://github.com/wnpower/NodeJS-TestApp.git
	cd NodeJS-TestApp
	npm install ws
	node test_websockets.js
